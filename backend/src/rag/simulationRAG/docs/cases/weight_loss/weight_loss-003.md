인해야겠습니다. 따라서 문진 중에 획빵1씨 말고 환자의 주소를 들은
후 바로 키와몸무게를 물어 보고 이를 판단하는 것이 자연스런 진료가 되겠습니다.

계(COPD, Tb) 등을확인해야 하며, 고령의 환자일 경우 암의 가능성도 놓치면 안 되겠습니다.
