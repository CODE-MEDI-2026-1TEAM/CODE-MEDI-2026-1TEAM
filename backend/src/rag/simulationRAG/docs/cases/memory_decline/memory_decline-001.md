### 가능성이 높은 진단

3. 가성 치매

### 계획된 검사
1. 정신상태검人t
2. 뇌 MRI

### 예상 술기
1. 설명동의서 받기 : 알츠하이머 치매 등 감별을 위한 뇌 MRI/CT를 시행하기 위해
설명동의서를 받습니다.
2. 혈압 측정 : 고혈압력과 약물 복용 중이므로 진료실에서 혈압을 측정해 봅니다.
기억력장애는 기억력장애의 양상이 어떤지 character를꼭 짚고 넘어
가야 하고, 기억력장애가 갑자기 생겼는지 서서히 나빠졌는지도 감별
에 도움을 줄 수 있습니다.
병력 청취가끝나고 간이 MMSE를 해야하는데 적어도 2분은 걸릴 것을 예상해야하며, 환자가
대답이 느릴 경우 어느 정도 빠르게 치고 나가야 할수도 있습니다.
신체 검진을 꼼꼼히 하는 것을 포기하고 간략하게 꼭 필요한 것들만 하는 것도 좋습니다.
경도인지장애 환자일 경우 지금부터 생활습관을 교정하는 것이 알츠하이머로 이행하는 것을
늦출 수 있음을 교육하고«환자에게 알맞은 취미 생활을 추천해 줍니다.
