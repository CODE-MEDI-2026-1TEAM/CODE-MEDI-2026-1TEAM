다. 어떨 때 어지럼이 악화되고 어떨 때 호전되는지가 감별에 중요한 힌트가될 수도 있습니다.

되는 검사법들의 시행 방법과의미에 대해서 공부하고 넘어 갑니다.
대개 호전되는 경우가 많으니 안심시켜 주고, 어지럼증이 나타날 때 안정된 곳에서 휴식을 취해
2차적인 외상을 방지하도록 주의시킵니다.
